3D Buttons

Use the attribute "data-btn-type" on divs or block elements to get this premade 3D-looking button partially inspired by Gimkit's GUI.
"data-btn-fade" sets a color transition on clicking the button.
"data-btn-smooth" is a boolean value for enabling/disabling smooth color transitions (It is true by default).
Below is a list of colors and options available for the 3D buttons.
