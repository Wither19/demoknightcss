The Vocabulary Tag <vocab>

This element is inline and makes text inside of it lime-colored and underlined. It inverts colors on hover.

You should use this when you have a word or term you need emphasized


Usage

 
Trust me, this is an important word. Are you ready? Here we go... <vocab>Tapioca</vocab>

Result
Trust me, this is an important word. Are you ready? Here we go...Tapioca
